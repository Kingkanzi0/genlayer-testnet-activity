# GenLayer Bradbury Testnet Activity

## Overview
My contributions as a **Builder** on GenLayer Bradbury Testnet using GenLayer Studio.

## Deployed Intelligent Contracts

### 1. Football Prediction Market
- File: `football_prediction_market.py`
- Description: A real-world prediction market contract for football matches.
- Features: Can automatically resolve using web data + AI after the match ends.
- Match: Czechia vs Mexico (2026-06-25)

### 2. Log Indexer (AI Vector Search)
- File: `log_indexer.py`
- Description: Semantic search engine using vector embeddings.
- Features: Uses AI to find similar logs by meaning (not just keywords). Achieved 0.995+ similarity scores.

## Technologies Used
- Intelligent Contracts written in Python
- On-chain LLM / AI reasoning
- Vector Embeddings (all-MiniLM-L6-v2)
- Web access for real-world data

## Repository Purpose
Showcasing practical usage of GenLayer’s AI-native blockchain features on the Bradbury Testnet.
